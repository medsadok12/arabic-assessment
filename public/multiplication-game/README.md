# 🧮 آيات — تعلّم الحساب باللعب

لعبة تعليمية **بسيطة لكن ذكية** لتعلّم **الضرب والقسمة والجمع والطرح** (المدى ١–١٠) للأطفال.
عند الدخول يجد الطفل أربع خانات واضحة: ✖️ ضرب · ➗ قسمة · ➕ جمع · ➖ طرح.
من إنتاج **أكاديمية عارم** — نتعلّم باللعب 💜

تطبيق **مستقل بالكامل**: بلا تبعيات، بلا خطوة بناء، ويعمل **أوفلاين**.
وهو **PWA** قابل للتثبيت على الشاشة الرئيسية للجهاز اللوحي كتطبيق حقيقي.

---

## 📲 التثبيت على جهاز لوحي (للأطفال)

بعد نشر المجلد على رابط (GitHub Pages أو Vercel)، افتح الرابط على الجهاز اللوحي ثم:

- **آيباد (Safari):** زر المشاركة ⬆️ ← «إضافة إلى الشاشة الرئيسية».
- **أندرويد (Chrome):** القائمة ⋮ ← «تثبيت التطبيق» أو «إضافة إلى الشاشة الرئيسية».

ستظهر **أيقونة ×** على الشاشة الرئيسية، يفتح التطبيق **بملء الشاشة** و**يعمل أوفلاين** بعد أول فتح.

---

## 🚀 التشغيل (للتطوير)

افتح `index.html` في أي متصفّح — وهذا كل شيء.

أو شغّل خادماً محلياً بسيطاً (اختياري):

```bash
cd public/multiplication-game
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```

> ملاحظة: خدمة العامل (offline) تعمل فقط عبر `http(s)`، لا عبر فتح الملف مباشرةً (`file://`).

## 🧠 لماذا «ذكية»؟

الذكاء ليس في الرسوم، بل في أن النظام **يعرف ماذا يعرف الطفل** ويكيّف نفسه:

1. **التكرار المتباعد (Leitner):** لكل حقيقة ضرب «صندوق» من ٠ إلى ٥. الإجابة الصحيحة ترفعها، والخطأ يُعيدها للأسفل. الحقائق الضعيفة تظهر **أكثر** حتى يُتقنها الطفل.
2. **اختيار أسئلة مُرجَّح:** الأسئلة الجديدة والضعيفة لها أولوية أعلى — لا إضاعة وقت فيما أُتقِن.
3. **فتح تدريجي للعوالم:** يبدأ بالجداول السهلة (×١، ×١٠، ×٢، ×٥…) ويفتح الأصعب عند الإتقان.
4. **خريطة إتقان ١٠×١٠:** صورة فورية ملوّنة (أخضر/أصفر/أحمر) لكل حقيقة — مفيدة للطفل والأهل والمعلّم.

## 🎮 لماذا «لعبة»؟

- ⭐ **نجوم** لكل عالم (حسب الدقّة) و**سلاسل (streak)** مع احتفال.
- 🎊 **أصوات وأنيميشن** (الأصوات مولّدة بـ Web Audio — بلا ملفات).
- 🦉 **تميمة تتفاعل** مع كل إجابة.
- 💡 **تلميح بصري:** يعرض الضرب على شكل صفوف من النقاط لبناء الفهم لا الحفظ الأعمى.
- ثلاثة أوضاع لعب:
  | الوضع | الوصف |
  |------|--------|
  | 🗺️ **المغامرة** | اكتشف العوالم جدولاً جدولاً، واجمع النجوم |
  | 🧠 **تدريب ذكي** | يركّز على ما تحتاج تدريبه (التكرار المتباعد) |
  | ⚡ **تحدّي الزمن** | كم إجابة صحيحة في ٦٠ ثانية؟ |

### 🔢 أكمل النمط (تنمية التفكير)
نشاطٌ خامس مستقلّ في الرئيسية لتقوية التفكير الرياضي: تُعرض متتالية بفراغ
(`2 → 4 → 6 → 8 → ⬜`) ويكتشف الطفل القاعدة ويُكملها. الصعوبة **تتكيّف**
تلقائياً (٥ مستويات): زيادة/نقصان ثابت، ثم مضاعفة (`×2`, `×3`)، وأحياناً فراغ
في الوسط. زرّ «ساعدني أفهم» يكشف القاعدة. تقدّم ونجوم مستقلّة.

### 🧩 الأحاجي (الاستدلال)
ألغاز عددية بصعوبة متكيّفة: **العدد الغامض** (`⬜ + 4 = 11`، `12 ÷ ⬜ = 3`)
و**ألغاز لفظية** («أنا عددٌ، إذا ضربتَني في ٣ أصير ١٢…»). تنمّي التفكير العكسي
وبذور الجبر. زرّ «ساعدني أفهم» يعطي تلميحاً للطريق.

### ⭐ النقاط والمستوى (نظام موحّد)
كل إجابة صحيحة في **أي نشاط** (العمليات + النمط + الأحاجي) تكسب نقاطاً
(١٠ للعمليات، ١٢ للنمط، ١٥ للأحاجي، +٥ للسلسلة). كل **١٠٠ نقطة = مستوى
جديد** باحتفال، مع لقب يتدرّج (مبتدئ ← ماهر ← بطل ← خبير ← عبقري). يظهر
المستوى وشريط النقاط في الرئيسية — يربط كل النشاطات بتقدّمٍ واحد متناسق.

> في كل التدريبات: لا انتقال إلا بعد حلّ السؤال صحيحاً (الخطأ لا يكشف الإجابة).

## 📁 البنية

```
public/multiplication-game/
  index.html      ← الهيكل والشاشات
  styles.css      ← التصميم (مبهج، متجاوب، RTL)
  game.js         ← المحرّك: التكرار المتباعد + التكيّف + اللعب + الأصوات
  manifest.json   ← بيانات الـ PWA (الاسم، الأيقونات، ملء الشاشة)
  sw.js           ← خدمة العامل (التشغيل أوفلاين)
  icon-*.png      ← الأيقونات (512/192/180/32)
  README.md
```

> موضوع داخل `public/` ليُنسَخ تلقائياً إلى مُخرَج بناء Vite (`dist`)،
> فيُنشَر مع الموقع على GitHub Pages وVercel تحت المسار `/multiplication-game/`.

- **حفظ التقدّم:** `localStorage` (يبقى بين الجلسات على نفس الجهاز).
- **بلا backend** — مناسب للنشر على Vercel أو GitHub Pages كملفات ساكنة.
- **متجاوب** مع الهواتف والأجهزة اللوحية (أزرار كبيرة مناسبة للأطفال)، ويدعم لوحة المفاتيح الفعلية.
