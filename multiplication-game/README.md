# ✖️ بطل الضرب — مغامرة جدول الضرب

لعبة تعليمية **بسيطة لكن ذكية** لتعلّم جدول الضرب (الجداول ١–١٠) للأطفال.
من إنتاج **أكاديمية عارم** — نتعلّم باللعب 💜

تطبيق **مستقل بالكامل**: بلا تبعيات، بلا خطوة بناء، ويعمل **أوفلاين**.

---

## 🚀 التشغيل

افتح `index.html` في أي متصفّح — وهذا كل شيء.

أو شغّل خادماً محلياً بسيطاً (اختياري):

```bash
cd multiplication-game
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```

---

## 🧠 لماذا «ذكية»؟

الذكاء ليس في الرسوم، بل في أن النظام **يعرف ماذا يعرف الطفل** ويكيّف نفسه:

1. **التكرار المتباعد (Leitner):** لكل حقيقة ضرب «صندوق» من ٠ إلى ٥. الإجابة الصحيحة ترفعها، والخطأ يُعيدها للأسفل. الحقائق الضعيفة تظهر **أكثر** حتى يُتقنها الطفل.
2. **اختيار أسئلة مُرجَّح:** الأسئلة الجديدة والضعيفة لها أولوية أعلى — لا إضاعة وقت فيما أُتقِن.
3. **فتح تدريجي للعوالم:** يبدأ بالجداول السهلة (×١، ×١٠، ×٢، ×٥…) ويفتح الأصعب عند الإتقان.
4. **خريطة إتقان ١٠×١٠:** صورة فورية ملوّنة (أخضر/أصفر/أحمر) لكل حقيقة — مفيدة للطفل والأهل والمعلّم.

---

## 🎮 لماذا «لعبة»؟

- ⭐ **نجوم** لكل عالم (حسب الدقّة) و**سلاسل (streak)** مع احتفال.
- 🎊 **أصوات وأنيميشن** (الأصوات مولّدة بـ Web Audio — بلا ملفات).
- 🦉 **تميمة تتفاعل** مع كل إجابة.
- 💡 **تلميح بصري:** يعرض الضرب على شكل صفوف من النقاط لبناء الفهم لا الحفظ الأعمى.
- ثلاثة أوضاع لعب:
  | الوضع | الوصف |
  |------|--------|
  | 🗺️ **المغامرة** | اكتشف العوالم جدولاً جدولاً، واجمع النجوم |
  | 🧠 **تدريب ذكي** | يركّز على ما تحتاج تدريبه (التكرار المتباعد) |
  | ⚡ **تحدّي الزمن** | كم إجابة صحيحة في ٦٠ ثانية؟ |

## 📁 البنية

```
multiplication-game/
  index.html   ← الهيكل والشاشات
  styles.css   ← التصميم (مبهج، متجاوب، RTL)
  game.js      ← المحرّك: التكرار المتباعد + التكيّف + اللعب + الأصوات
  README.md
```

- **حفظ التقدّم:** `localStorage` (يبقى بين الجلسات على نفس الجهاز).
- **بلا backend** — مناسب للنشر على Vercel أو GitHub Pages كملفات ساكنة.
- **متجاوب** مع الهواتف والأجهزة اللوحية (أزرار كبيرة مناسبة للأطفال)، ويدعم لوحة المفاتيح الفعلية.

## 🧪 الاختبار

جرى التحقّق من المنطق بسيناريو دخان عبر `jsdom` (لعب جولة مغامرة كاملة، والتنقّل بين كل الأوضاع، ورصد أخطاء التشغيل). النتيجة: **٢٢ فحصاً ناجحاً، صفر أخطاء**.
